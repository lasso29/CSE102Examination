Array is Javascript method that is used store data and Information and it is denoted by [ ]

Array Methods

  • Push: It add to the end of element in an Array (add)
  • Pop: It removes the element at the end of an Array (remove)
  • Shift: It removes the first element at the begining of Array
  • Unshift: It add element at the begining of an Array to make the element first
  • Reverse: It arranges the element in an Array in descending order i.e (from back to front)
